// Case Study 003

Quantum Lab

Visualizing The
Subatomic

A visual quantum gate simulator designed for education and experimentation. The project bridges the gap between abstract quantum math and visual representation, allowing users to manipulate qubits across a Bloch sphere.

// Tech Stack

  • TypeScript
  • React (Zustand State)
  • Three.js (Rendering)
  • Tailwind CSS

// The Core Logic

Developed a custom high-performance **Mathematical Mesh** using Three.js to render real-time vector rotations on the Bloch sphere with sub-degree accuracy.
