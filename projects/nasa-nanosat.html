NASA Nanosat MO Framework

Contributed to NASA's open-source framework for satellite on-board software development, providing SDK, simulators, and examples for space technology.

Overview

NASA's Nanosat MO Framework is an open-source project that provides a complete software development kit for building satellite on-board software. The framework implements the Spacecraft Message Organization (Space MO) standard for spacecraft communication.

As a contributor, I helped develop SDK components, example applications, and simulator integrations that enable developers to build and test satellite software in a realistic environment.

Key Components

  • SDK: Complete development kit with APIs for satellite operations
  • OPS-SAT Simulator: Software-only spacecraft simulator for testing
  • Python Examples: Scripting examples for mission control
  • Picture Processor: Sample application for image handling
  • Edge AI Integration: Examples for running ML models on satellites
  • Digital Twins: Virtual satellite models for testing
