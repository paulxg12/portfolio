# Brand Identity: Prince Raymond Paul

## Overview

A minimal, sophisticated personal brand for a software engineer. Clean typography, muted tones, and timeless design that conveys technical excellence and refined taste.

---

### Primary: Display & Headers

**Font**: [Inter](https://fonts.google.com/specimen/Inter) or [DM Sans](https://fonts.google.com/specimen/DM+Sans)

| Element | Weight | Size | Letter-spacing |
|---------|--------|------|----------------|
| Name (Logo) | 300 (Light) | 48px | 0.2em |
| H1 | 600 (SemiBold) | 36px | 0.05em |
| H2 | 500 (Medium) | 24px | 0.03em |
| H3 | 500 (Medium) | 18px | 0.02em |

|---------|--------|------|-------------|

## Visual Language

### Spacing System

```
Base unit: 8px

xs:   4px   (0.5x)
sm:   8px   (1x)
md:  16px   (2x)
lg:  24px   (3x)
```

### Border Radius

```
None:  0px    (buttons, inputs - sharp)
sm:    4px    (cards)
md:    8px    (modals)
lg:    16px   (large containers)
```

## Brand Do's & Don'ts

### DO
- Use ample whitespace
- Keep it minimal - less is more
- Use Inter or DM Sans typography
- Apply warm gold sparingly as accent
- Maintain consistent letter-spacing

### DON'T
- Add gradients or 3D effects
- Use more than 2 fonts
- Overcrowd with elements
- Use bold/heavy weights excessively
- Apply accent colors liberally
